Updating

The updating instructions for ApPHP Framework are cumulative. That is, if you want to update from version A to version C and there is version B between A and C, you need to following the instructions for both A and B.

General update intructions

  1. Make a backup.
  2. Replace 'framework' dir with the new one or point SVN to a fresh release and update.
  3. Check if everything is OK, if not - revert from backup and post issues to ApPHP issue tracker.

Special update intructions

  1. May be found in the UPDATE file, placed in script archive.
