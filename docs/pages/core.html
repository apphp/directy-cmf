Framework Structure - Core

There are some base classes in ApPHP Framework also known as "core" classes. These classes release the main ideas and architecture of the framework. All these classes are found in framework/core/ directory.

Here the core classes:
  • framework/ framework directory
    • core/ directory where placed all core classes
      • CController.php CController base class
      • CDebug.php CDebug class
      • CModel.php CModel base class
      • CRouter.php CRouter class
      • CView.php CView base class
      • interfaces.php class with interfaces
